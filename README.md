# Sweet Shop Management System

Welcome to **Sweet Tooth**, an out-of-this-world Sweet Shop Management System. This full-stack application allows users to browse a galaxy of sweets, while administrators can manage the inventory with ease.


## Features

- Add, View, Delete Sweets
- Search by Name, Category, or Price Range
- Purchase & Restock with Stock Validation
- Responsive, Clean, User-Friendly UI

## Technologies Used

**Frontend**
- React.js
- Tailwind CSS (for styling)
- Vite (Build tool)

**Backend**
- Node.js
- Express.js
- MongoDB (Database)
## Deployment

**Backend**

```bash
  cd backend
  npm install
  npm start
```

**Frontend**
```bash
  cd frontend
  npm install
  node src/server.js
```
